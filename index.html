file_download 安装脚本 卸载脚本

Howto

  1. 下载安装脚本。如果你之前没有开启 WSL 功能，请先执行一次脚本。这将会开启 WSL。请重启计算机并继续
  2. 下载并安装 WSL2 Kernel Update
  3. 右键脚本，"使用 PowerShell 运行"。脚本会请求管理员权限
  4. 安装完成后可以使用 Windows Terminal 启动 WSL Distro，或者你可以直接执行 wsl.exe -d physics-data-wsl
  5. debian 用户的默认密码为 debian，请尽快使用 passwd 命令修改。
  6. 卸载时，直接执行卸载脚本即可
